# 文件过滤器不显示css文件问题

### 在浏览器中进入index.jsp页面的时候不加载css文件，在网上百度多次一直没有任何靠谱的答案。



### 后来发现原来是写文件过滤器的时候，不过滤文件只写了index.jsp和signin.jsp文件，而css文件被过滤了无法加载。

### 只要在文件过滤器中添加相应的css文件就会正常加载出来啦。
